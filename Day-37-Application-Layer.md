## What is the Application Layer?

The Application Layer is Layer 7 of the OSI Model.
It provides network services directly to end-user applications.

---

## What Does It Do?

- Enables web browsing
- Supports email communication
- Allows file transfers
- Handles domain name resolution

---

## Common Protocols

### HTTP / HTTPS
- Web communication
- Secure browsing (HTTPS)

### FTP
- File transfer protocol

### SMTP
- Email sending protocol

### DNS
- Resolves domain names to IP addresses

### Telnet / SSH
- Remote login access

---

## Security Importance

- Target for phishing attacks
- Vulnerable to SQL injection
- Cross-site scripting (XSS)
- Requires Web Application Firewall (WAF)
- Important for SOC monitoring
