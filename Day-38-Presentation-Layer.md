## What is the Presentation Layer?

The Presentation Layer is Layer 6 of the OSI Model.
It is responsible for formatting, encrypting, and compressing data before it reaches the Application Layer.

---

## Core Responsibilities

### Encryption
- Converts plaintext into ciphertext
- Protects sensitive information

### Decryption
- Converts ciphertext back to readable format

### Data Formatting
- Character encoding (ASCII, Unicode)
- File format conversion

### Compression
- Reduces data size
- Improves transmission efficiency

---

## Why It Matters in Cybersecurity

- Enables HTTPS encryption
- Protects login credentials
- Prevents data interception
- Important for secure web communication
- Used in SSL/TLS operations
