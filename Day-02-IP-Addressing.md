# Day 02 - IP Addressing Fundamentals

## Topics Covered

- IP address concept
- IPv4 vs IPv6
- Public IP vs Private IP

## Explanation

An IP address is a unique number assigned to a device on a network.

IPv4 uses 32-bit addressing while IPv6 uses 128-bit addressing.

Public IPs are used on the internet, while private IPs are used inside local networks.

## Why This Matters in Cybersecurity

Every attack, scan, and defense depends on IP addresses.

## My Key Takeaway

Without understanding IP addressing, cybersecurity has no direction.
