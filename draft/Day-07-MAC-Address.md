## What is MAC Address
MAC Address (Media Access Control Address) is a unique physical address assigned to a network interface card (NIC) by the manufacturer.

It is used to identify devices inside a local network.

## Format
A MAC address is 48 bits long and written in hexadecimal format.

Example:
00:1A:2B:3C:4D:5E

## Types of MAC Address
- Unicast
- Multicast
- Broadcast

## Why MAC Address is Important
MAC addresses help switches forward data correctly inside LAN.

Routers use IP addresses, but switches rely on MAC addresses.

## MAC vs IP
MAC Address:
- Physical
- Permanent
- Layer 2

IP Address:
- Logical
- Changeable
- Layer 3

## MAC Address in Cybersecurity
Attackers can spoof MAC addresses.
MAC filtering is used for basic access control.
MAC addresses help in network monitoring and investigation.

## My Key Takeaway
MAC address is the identity card of every network device.
